# Text Symbols

O programa utiliza `text-xxx` para mensagens. Abaixo estão sugestões de textos para cadastrar nos símbolos de texto do report.

| Símbolo | Texto sugerido |
|---|---|
| `E01` | Nenhum documento encontrado para os filtros informados. |
| `E02` | Erro ao converter dados para o formato CSV. |
| `E03` | Erro de controle ao abrir a janela de salvamento. |
| `E04` | Não há interface gráfica disponível para seleção do arquivo. |
| `E05` | Operação não suportada pela interface gráfica atual. |
| `E06` | Nome de arquivo padrão inválido. |
| `E07` | Erro inesperado ao definir o caminho do arquivo. |
| `E08` | Operação cancelada pelo usuário. |
| `E09` | Download não disponível em execução background ou ambiente sem GUI. |
| `E10` | Usuário sem autorização ou acesso negado ao caminho selecionado. |
| `E11` | Erro ao gravar arquivo, arquivo não encontrado ou disco cheio. |
| `E12` | Erro inesperado ao gerar o arquivo CSV. |
| `S01` | Arquivo CSV gerado com sucesso. |
